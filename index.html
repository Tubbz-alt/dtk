Deepin Tool Kit Widget

Deepint Tool Kit (Dtk) is the base devlopment tool of all C++/Qt Developer work on Deepin.

Dependencies

Build dependencies

  • Qt >= 5.6

Installation

Build from source code

  1. Make sure you have installed all dependencies.
  2. Build:
1 $ mkdir build
2 $ cd build
3 $ qmake ..
4 $ make
  1. Install:
1 $ sudo make install
