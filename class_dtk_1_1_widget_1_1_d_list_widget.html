bool toggleable
 这个属性表示列表控件的所有项的选择状态是否可以切换。 当设置为 true 时，鼠标点击某一个列表项，会根据当前的列表项的选择状态切换，如果当前是已选择状态则点击后切换到未选择状态，反之则反，此选项默认为 false 。 更多...
 
bool enableHorizontalScroll

详细描述

基于widget的列表控件。

DListWidget是一个自定义的列表控件，与QListWidget相比它虽然也是基于item的一个列表控件但却更灵活，因为它可以设置任意QWidget类型的对象作为自己的item，同时提供了各种新增，插入，移除item的方法以及信号，另外还支持设置选择模式为单选或多选。

bool Dtk::Widget::DListWidget::toggleable
readwrite

这个属性表示列表控件的所有项的选择状态是否可以切换。 当设置为 true 时，鼠标点击某一个列表项，会根据当前的列表项的选择状态切换，如果当前是已选择状态则点击后切换到未选择状态，反之则反，此选项默认为 false 。
